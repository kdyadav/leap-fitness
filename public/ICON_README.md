# PWA Icons

Replace the placeholder icons with your actual app icons:

- `pwa-192x192.png` - 192x192px PNG icon
- `pwa-512x512.png` - 512x512px PNG icon

You can create these icons using:

1. Online tools like https://favicon.io/ or https://www.pwabuilder.com/
2. Design software like Figma, Photoshop, or Canva
3. Use your app logo/branding

Recommended: Create a square icon with your app logo centered, using your brand colors.
